```html
<style>
    iframe {
        width: 1024px;
        height: 768px;
        border: none;
    }
</style>
```

Inline CSS for example purposes only of course. Feel free to pick a different size for the `iframe`, aspect ratio is 4:3. The canvas inside the `embed.html` will scale to whatever size you pick for the `iframe`.
